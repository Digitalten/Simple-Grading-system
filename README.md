# Grade Checker in C

Simple program in C that analyzes the user's grade and informs whether he passed or failed.

## Features
- Grade entry (0 to 10)
- Invalid value validation
- Pass/fail check

## Technologies
- C language

## How to execute
1. Compile:
   gcc notes.c -o notes

2. Run:
   ./notes
